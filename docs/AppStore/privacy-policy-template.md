# Channel Timeline Viewer プライバシーポリシー

最終更新日: [YYYY-MM-DD]

## 1. はじめに
Channel Timeline Viewer（以下「本アプリ」）は、チャンネルの投稿動画を公開日順に整理して視聴を補助する
アプリです。本ポリシーは本アプリが扱う情報について説明します。

## 2. 収集・保存する情報
本アプリは、運営者のサーバーに利用者の個人情報を送信・収集しません。

- **端末内にのみ保存する情報**（外部送信なし）:
  - 視聴済みにした動画の識別子（videoId）と日時
  - お気に入り（最近使った）チャンネルの情報（チャンネルID・名称・サムネイルURL・最終利用日時）
  - これらは iOS の UserDefaults に保存され、アプリを削除すると消去されます。

## 3. 第三者サービスの利用
本アプリは以下の YouTube（Google LLC）公式サービスを利用します。

- **YouTube Data API v3**: 入力されたチャンネルの公開された動画一覧情報を取得します。
- **YouTube IFrame Player（埋め込みプレイヤー）**: 動画の再生に使用します。再生時、YouTube/Google が
  独自のポリシーに基づき情報を取得・処理する場合があります。詳細は Google のプライバシーポリシーをご確認ください。
  - Google プライバシーポリシー: https://policies.google.com/privacy

本アプリは動画のダウンロード、広告の回避、スクレイピング、バックグラウンド再生を行いません。

## 4. トラッキング
本アプリは広告識別子（IDFA）を使用したトラッキングを行いません。

## 5. 子どものプライバシー
本アプリは特定の年齢層を対象に個人情報を収集することはありません。

## 6. ポリシーの変更
本ポリシーは必要に応じて更新されることがあります。重要な変更がある場合はアプリの更新等でお知らせします。
