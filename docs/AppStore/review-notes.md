# App 審査向けメモ（App Review Notes）

App Store Connect の「App Review Information > メモ（Notes）」に記載する内容の下書きです。

---

## アプリの目的
本アプリはチャンネルの投稿動画を公開日順（古い順）に整理し、順番に視聴しやすくする**視聴補助アプリ**です。
YouTube の代替アプリではありません。

## 動画一覧の取得
- 取得は **YouTube Data API v3**（公式 API）のみを使用します。スクレイピングは行いません。
- 取得する情報: videoId / タイトル / 説明 / 公開日 / サムネイルURL。

## 動画の再生
- 再生は **YouTube IFrame Player API（公式の埋め込みプレイヤー）** を WKWebView 上で表示します。
- 動画ファイルの取得・保存・独自プレイヤーでの再生は行いません。

## 行っていないこと（ポリシー順守）
- 動画のダウンロード: **なし**
- スクレイピング: **なし**（公式 API のみ）
- 独自プレイヤーでの動画再生: **なし**（公式 IFrame Player のみ）
- 広告の回避・ブロック: **なし**
- バックグラウンド再生: **なし**（`UIBackgroundModes` 未使用、`AVAudioSession`/`AVPlayer` 不使用）

> 補足: ソース上も `AVPlayer` / `AVAudioSession` / `UIBackgroundModes` / ダウンロード処理 / `.mp4` 取得 /
> スクレイピング用エンドポイントは含まれていません（再生は `loadHTMLString` による公式 IFrame 埋め込みのみ）。

## レビュー時のテスト手順
1. アプリ起動後、チャンネルURLを入力（例: 任意の公開チャンネルの `https://www.youtube.com/@handle`）。
2. 「動画を取得」をタップ → 投稿動画が公開日順（古い順）で一覧表示されます。
3. 動画をタップ → 公式埋め込みプレイヤーで再生。「前へ」「次へ」で移動できます。
4. 「ⓘ このアプリについて」に注意事項（公式アプリではない 等）を表示しています。

## API キーについて（重要）
- 動画一覧の取得には YouTube Data API v3 の API キーが必要です。
- **提出ビルドには有効な API キーを含めてください**（`Resources/Config.plist` の `YOUTUBE_API_KEY`）。
  キーが無いと一覧取得時に「APIキーが設定されていません」と表示され、レビューで機能を確認できません。
- 必要に応じて、レビュー用のテスト手順・サンプルチャンネルURLをこのメモに追記してください。
