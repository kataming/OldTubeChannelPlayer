## アプリ名（表示名）
Channel Timeline Viewer

## サブタイトル（30字以内目安）
チャンネル動画を公開日順で視聴

## プロモーションテキスト（170字以内）
気になるチャンネルの動画を「古い順」に並べて、最初から順番に視聴。公式の埋め込みプレイヤーで再生し、視聴済み管理やお気に入りチャンネル保存もできます。

## 説明文（日本語）

Channel Timeline Viewer は、チャンネルの投稿動画を「公開日順（古い順）」に整理して、最初の動画から
順番に視聴しやすくする視聴補助アプリです。

シリーズものや解説動画、講座などを「古い順に1本ずつ追っていきたい」ときに便利です。

■ 主な機能
・チャンネルのURLを入力して、投稿動画を公開日順（古い順／新しい順）に一覧表示
・サムネイル・タイトル・公開日つきの見やすいリスト
・タップして順番に再生（「前へ」「次へ」で移動）
・視聴済みマークで「どこまで見たか」を管理
・お気に入り（最近使った）チャンネルの保存

■ 再生について
動画の再生には YouTube 公式の埋め込みプレイヤー（IFrame Player）を使用します。

■ ご注意
・このアプリは YouTube 公式アプリではありません。
・動画の再生には YouTube 公式の埋め込みプレイヤーを使用しています。
・動画のダウンロード、広告の回避、バックグラウンド再生は行いません。
・動画一覧の取得には YouTube Data API v3 を使用します。

※ YouTube は Google LLC の商標です。

## 説明文（英語）

Channel Timeline Viewer helps you watch a channel's uploads in chronological (oldest‑first) order,
so you can follow a series, course, or explainer from the very beginning.

Features
• Enter a channel URL and list its uploads by publish date (oldest / newest first)
• Clean list with thumbnails, titles, and publish dates
• Tap to play in order, with Previous / Next controls
• "Watched" marks to track your progress
• Save favorite (recently used) channels

Playback
• Videos play using YouTube's official embedded player (IFrame Player).

Please note
• This app is NOT the official YouTube app.
• Playback uses YouTube's official embedded player.
• It does NOT download videos, block ads, or play in the background.
• Video lists are retrieved via the YouTube Data API v3.

YouTube is a trademark of Google LLC.
