## サブタイトル（30字以内目安）

## プロモーションテキスト（170字以内）
気になるチャンネルの動画を公開日順（古い順）に整理。最初から順番に視聴でき、視聴済み管理・進捗率・「続きから見る」・
動画メモで、シリーズ視聴や学習の進み具合を管理できます。再生は YouTube 公式の埋め込みプレイヤーを使用します。

## 説明文（日本語）

Channel Timeline Viewer は、チャンネルの投稿動画を「公開日順（古い順）」に整理し、最初の動画から順番に
視聴・学習しやすくする視聴補助アプリです。解説シリーズ・講座・連載のような動画を「最初から順番に」追うのに向いています。

■ こんな人に
・チャンネルの過去動画を最初から順番に見たい
・学習教材のように「どこまで見たか」を管理したい
・シリーズものを抜け漏れなく視聴したい

■ 主な機能
・チャンネルURLを入力 → 投稿動画を公開日順（古い順／新しい順）に一覧表示
・チャンネルごとの進捗（視聴済み数 / 総本数・進捗率）と進捗バー
・「次に見る／続きから見る」導線で、続きをすぐ再生
・フィルター（すべて／未視聴のみ／視聴済みのみ）
・視聴済みチェック、複数チャンネルのお気に入り管理
・動画ごとのメモ（学習・シリーズ視聴用）

■ 再生について
動画の再生には YouTube 公式の埋め込みプレイヤー（IFrame Player）を使用します。

■ ご注意
・本アプリは YouTube 公式アプリではありません。
・動画の再生には YouTube 公式の埋め込みプレイヤーを使用しています。
・動画のダウンロードは行いません。
・広告回避は行いません。
・バックグラウンド再生は行いません。
・動画一覧の取得には YouTube Data API v3 を使用します。

※ YouTube は Google LLC の商標です。

## 説明文（英語）

Channel Timeline Viewer helps you watch and study a channel's uploads in chronological (oldest‑first) order —
ideal for following a course, explainer series, or long-running series from the very beginning.

Great for
• Watching a channel's back catalog from the start
• Tracking how far you've watched, like a learning course
• Following a series without missing episodes

Features
• Enter a channel URL → list uploads by publish date (oldest / newest first)
• Per-channel progress (watched / total, percentage) with a progress bar
• "Next / Continue" to resume quickly
• Filters: All / Unwatched only / Watched only
• Watched marks, multiple favorite channels
• Per-video notes for studying or series viewing

Playback
• Videos play using YouTube's official embedded player (IFrame Player).

Please note
• This app is NOT the official YouTube app.
• Playback uses YouTube's official embedded player.
• It does NOT download videos, block ads, or play in the background.
• Video lists are retrieved via the YouTube Data API v3.
