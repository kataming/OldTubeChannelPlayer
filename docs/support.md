# Channel Timeline Viewer サポート

Channel Timeline Viewer は、チャンネルの投稿動画を公開日順（古い順）に整理し、過去動画視聴・シリーズ視聴・
学習を進捗管理しながら行うための**視聴補助アプリ**です。**YouTube 公式アプリではありません。**

## よくある質問

**Q. 動画が再生できません / プレイヤーが「設定エラー」になります**
A. 動画は YouTube 公式の埋め込みプレイヤーで再生します。通信環境をご確認ください。一部の動画は権利者の設定で
埋め込み再生が許可されていない場合があります。その場合は「YouTubeで開く」をご利用ください。

**Q. 動画一覧が取得できません / 「APIキーが設定されていません」と出ます**
A. 本アプリは YouTube Data API v3 を使用します。お使いのビルドに API キーが設定されている必要があります。

**Q. 視聴履歴・メモはどこに保存されますか**
A. すべて端末内（iOS の保存領域）にのみ保存され、外部送信されません。アプリを削除すると消去されます。

**Q. ダウンロードやバックグラウンド再生はできますか**
A. 行いません。本アプリは動画のダウンロード・広告回避・バックグラウンド再生を行いません。
