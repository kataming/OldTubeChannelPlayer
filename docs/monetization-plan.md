# 収益化方針（Monetization Plan）

> 本ドキュメントは**方針の整理**であり、課金実装はまだ行わない。
> YouTube の規約・App Store のガイドラインに反しない形を前提とする。

## 基本方針（初回リリース）
- **無料**で提供する。
- **広告なし**（広告SDK・バナー・インタースティシャル等を一切入れない）。
- **アプリ内課金（IAP）なし**。
- **YouTube 動画の再生そのものには課金しない**（再生は YouTube 公式埋め込みプレイヤーで提供）。
- 以下の「規約に抵触しうる機能」は**提供しない**（有料・無料を問わず）:
  - 広告なし再生 / 広告のスキップ・回避
  - バックグラウンド再生
  - 動画のダウンロード / オフライン保存
  - YouTube の再生制限・UI の回避

> ねらい：本アプリは「YouTube 視聴の代替・特典提供」ではなく、**チャンネル動画を時系列で整理し、視聴の進捗を
> 管理する補助ツール**。課金するとしても、その対象は **YouTube コンテンツではなくアプリ独自の機能**に限定する。

## 将来的な Pro 版の考え方
Pro 版（有料）を導入する場合でも、課金対象は **アプリ独自の整理・進捗管理機能**とする。
**YouTube のコンテンツ・再生体験（広告なし/BG/DL 等）に対しては課金しない。**

### Pro 版候補（アプリ独自機能のみ）
- お気に入りチャンネルの**無制限**保存（無料版は上限あり）
- 動画メモの**無制限**保存（無料版は上限あり）
- **複数リスト**の作成（自分用の視聴リストを複数管理）
- 進捗データの**エクスポート**（CSV/JSON 等。視聴済み・進捗の書き出し）
- **高度なフィルター**（複合条件・並び替えの保存など）
- **iCloud 同期 / バックアップ**（端末間での進捗・お気に入り・メモの同期）

> いずれも「アプリが提供する整理・管理の利便性」への対価であり、YouTube 動画そのものの提供・再生特典ではない。

### Pro 版を実装する際の留意点（将来）
- StoreKit 2 による IAP（サブスク or 買い切り）として実装。
- 無料版の既存機能は維持し、Pro はあくまで上限解放・追加機能とする。
- App Store のガイドライン（課金は Apple の IAP を使用 等）に従う。
- iCloud 同期を入れる場合、保存データ・同期範囲をプライバシーポリシーに追記する。
- 実装時はこの方針の範囲（YouTube コンテンツに課金しない）を再確認する。

## 広告方針
- **初回リリースでは広告を入れない。**
- **将来的にも、再生画面およびその周辺には広告を表示しない**（公式プレイヤー体験に広告を絡めない）。
- 仮に将来どこかに広告を導入する場合は、導入前に必ず:
  - **プライバシーポリシー**（[`privacy-policy.md`](privacy-policy.md)）を更新（広告SDK・収集データ・トラッキングの有無）
  - **App Store のデータ申告（App Privacy）**を更新（[`app-store-connect-fields.md`](app-store-connect-fields.md)）
  - 必要に応じて ATT（App Tracking Transparency）対応を追加
- 広告を入れる場合も、YouTube の再生周辺・公式プレイヤーに重ねる形は採らない。

## まとめ（リリース時点のスタンス）
| 項目 | 初回リリース | 将来 Pro（候補） |
|---|---|---|
| 価格 | 無料 | 一部機能を有料化（アプリ独自機能のみ） |
| 広告 | なし | なし（少なくとも再生周辺は恒久的になし） |
| IAP | なし | お気に入り/メモ無制限・複数リスト・エクスポート・高度フィルター・iCloud同期 等 |
| YouTube 再生への課金 | しない | しない |
| DL/広告回避/BG再生 | 提供しない | 提供しない |

---
※ 本方針は実装前のドキュメント。実際の課金・広告を導入する際は、YouTube API サービス利用規約・
App Store Review ガイドライン・各種ポリシーを再確認すること。
