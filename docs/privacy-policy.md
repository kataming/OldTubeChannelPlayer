# プライバシーポリシー

最終更新日: [YYYY-MM-DD]

## 1. はじめに
Channel Timeline Viewer（以下「本アプリ」）は、チャンネルの投稿動画を公開日順（古い順）に整理し、
過去動画視聴・シリーズ視聴・学習を進捗管理しながら行うための**視聴補助アプリ**です。
**本アプリは YouTube 公式アプリではありません。**

## 2. 収集・保存する情報
本アプリは、運営者のサーバーに利用者の個人情報を送信・収集しません。

- **端末内にのみ保存する情報**（外部送信なし・iOS の UserDefaults）:
  - 視聴済みにした動画の識別子（videoId）と日時
  - チャンネルごとの進捗（総本数・視聴済み数・最後に開いた動画）
  - お気に入り（最近使った）チャンネル情報（チャンネルID・名称・サムネイルURL・最終利用日時）
  - 動画ごとのメモ
  - これらはアプリを削除すると消去されます。

## 3. 第三者サービスの利用（YouTube / Google）
本アプリは以下の YouTube（Google LLC）公式サービスを利用します。

- **YouTube Data API v3**：入力されたチャンネルの**公開された**動画一覧情報の取得に使用します。
- **YouTube IFrame Player（公式の埋め込みプレイヤー）**：動画の再生に使用します。再生時、YouTube/Google が
  独自のポリシーに基づき情報を取得・処理する場合があります。

本アプリは、動画のダウンロード、広告の回避、スクレイピング、バックグラウンド再生、独自プレイヤーでの再生を
**一切行いません**。

YouTube/Google の取り扱いについては、各社のポリシー・規約をご確認ください。
- Google プライバシーポリシー: https://policies.google.com/privacy
- YouTube 利用規約: https://www.youtube.com/t/terms

## 4. トラッキング
本アプリは広告識別子（IDFA）を用いたトラッキングを行いません（App Tracking Transparency 不使用）。

## 5. 子どものプライバシー
本アプリは特定の年齢層を対象に個人情報を収集することはありません。

## 6. ポリシーの変更
本ポリシーは必要に応じて更新されることがあります。重要な変更時はアプリ更新等でお知らせします。
