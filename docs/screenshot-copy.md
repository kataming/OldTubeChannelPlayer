# App Store スクリーンショット用 文言（キャプション）

> 各スクショに重ねる短いキャッチコピー案。位置づけは「時系列視聴・進捗管理・学習補助」。
> **YouTube ロゴ・ブランド要素を画面外に重ねない／「公式」と誤認させない**こと（再生画面内に出る公式プレイヤーUIはそのまま）。

## 推奨スクショ構成（6.7インチ等・縦）

1. **ホーム / お気に入り**
   - 見出し：「チャンネルを“最初から順番に”」
   - サブ：「お気に入りごとに進捗を管理」
2. **動画一覧（進捗バー＋次に見る）**
   - 見出し：「古い順で、続きから」
   - サブ：「12 / 120本 視聴済み・次に見る動画がひと目で」
3. **フィルター（未視聴のみ）**
   - 見出し：「未視聴だけ表示」
   - サブ：「見るべき動画をすぐ見つける」
4. **再生画面（公式プレイヤー＋メモ）**
   - 見出し：「公式プレイヤーで再生＋メモ」
   - サブ：「学習・シリーズ視聴の要点を動画ごとに記録」
5. **進捗 / 続きから**
   - 見出し：「学習教材のように進捗管理」
   - サブ：「進捗率・続きから見るで挫折しない」
6. **About / 注意事項**（任意）
   - 見出し：「公式アプリではない補助ツール」
   - サブ：「再生は公式埋め込み・DL/広告回避/BG再生なし」

## キャッチコピー候補（ストア上部・1行）
- 「チャンネル動画を“最初から順番に”視聴・学習」
- 「古い順 × 進捗管理で、過去動画を最後までやり切る」
- 「シリーズ視聴・学習のための時系列ビューア」

## 表現の注意（規約・審査）
- 「YouTube を見るアプリ」ではなく **「チャンネル動画を時系列で整理・進捗管理する補助アプリ」** と表現する。
- 「公式」「YouTube公式」等、公式と誤認させる文言を使わない。
- YouTube のロゴ・赤い再生ボタン等のブランド要素をキャプションやアイコンに使わない（公式プレイヤー内の表示は除く）。
- 「ダウンロード」「広告なし」「バックグラウンド再生」など、本アプリが行わない機能をうたわない。
- 必要に応じて末尾に「YouTube は Google LLC の商標です」と記載。
