# 実画面確認チェックリスト（Mac / Xcode / 実機・シミュレーター）

GitHub Actions ではコンパイルとユニットテストまで確認済み。ここでは **実際の画面操作**を Mac/Xcode で確認する。
Windows では確認できない「見た目・操作・実再生」を埋めるためのもの。

## 事前準備
- [ ] Mac で `brew install xcodegen`
- [ ] リポジトリ取得後、`xcodegen generate` → `ChannelTimelineViewer.xcodeproj` が生成される
- [ ] `Resources/Config.plist` を作成し、実際の `YOUTUBE_API_KEY` を設定（[`app-store-preparation.md`](app-store-preparation.md) 参照）
- [ ] Xcode でプロジェクトを開き、ターゲット `ChannelTimelineViewer` をシミュレーター（iPhone 15等）で実行（Cmd+R）
  - ※ シミュレーターなら署名・Apple Developer 登録は不要

---

## 1. ホーム画面（ChannelInputView）
- [ ] アプリ名・チャンネルURL入力欄が表示される
- [ ] 右上に「ⓘ」ボタンがある → タップで「このアプリについて」が開く
- [ ] APIキー未設定時：「APIキーが設定されていません」の警告が出る（Config.plist 未作成で確認）
- [ ] 画面下部に注意事項（公式アプリではない等）が表示される
- [ ] チャンネルURL（`@handle` / `/channel/UC...`）を入力して「動画を取得」→ 一覧画面へ遷移する
- [ ] 不正なURLや存在しないチャンネルで、日本語のエラーメッセージが出る

## 2. このアプリについて（AboutView）
- [ ] アプリ名・用途説明が出る
- [ ] 注意事項が **5点**表示される（公式でない／公式埋め込み／DLしない／広告回避しない／BG再生しない）
- [ ] `Config.plist` に `PRIVACY_POLICY_URL` を設定した場合、プライバシーポリシーのリンクが出てタップで開く
- [ ] 「閉じる」で戻れる

## 3. 動画一覧画面（VideoListView）
- [ ] チャンネル名がタイトルに出る
- [ ] **進捗バー**と「X / Y本 視聴済み（Z%）」が表示される
- [ ] **「次に見る：第N本目」**（または「続きから：第N本目」）が目立つ位置に表示され、タップで再生画面へ
- [ ] 右上のフィルターメニューで：
  - [ ] 並び替え「古い順 / 新しい順」が切り替わり、リスト順が変わる
  - [ ] 表示「すべて / 未視聴のみ / 視聴済みのみ」が切り替わり、リストが絞り込まれる
- [ ] ヘッダーに「N本表示 / 全M本」が出る
- [ ] 各行にサムネ・タイトル・公開日・（視聴済みなら）緑チェックが出る
- [ ] すべて視聴済みにすると「すべて視聴済みです 🎉」になる

## 4. 再生画面（PlayerView）
- [ ] YouTube 公式埋め込みプレイヤーが表示され、再生できる
- [ ] 「前へ / 次へ」で動画が切り替わる（端で無効化される）
- [ ] 「視聴済みにする / 視聴済みを解除」が切り替わり、一覧の緑チェック・進捗に反映される
- [ ] **メモ欄**に日本語入力でき、画面遷移して戻っても保存されている（別動画では別メモ）
- [ ] 「YouTubeで開く」で公式アプリ/サイトが開く
- [ ] 再生終了時に「次の動画」候補が出る（endedイベント取得時。取れない場合は「次へ」で手動でOK）

## 5. 進捗・続きから（横断）
- [ ] 動画を何本か視聴済みにし、ホームに戻ると、お気に入りのチャンネル行に進捗バーと「続きから見る」が出る
- [ ] チャンネルを再度開くと「続きから：第N本目」が前回の続き（最後に開いた未視聴 or 最古未視聴）を指す
- [ ] 「未視聴のみ」フィルターで、次に見るべき動画が探しやすい

## 6. お気に入り（複数チャンネル）
- [ ] 複数チャンネルを取得すると、ホームに最近使った順で複数並ぶ
- [ ] 各チャンネルに進捗率が出る
- [ ] スワイプ（または編集）でチャンネルを削除でき、進捗も消える

## 7. 規約順守の目視確認
- [ ] 動画は必ず公式プレイヤー内で再生される（独自プレイヤー化していない）
- [ ] ダウンロードボタン等が無い
- [ ] バックグラウンド（ホームに戻る/画面ロック）で音声が止まる（BG再生していない）

---

## 不具合があった場合
- 画面・操作の不具合は Xcode のコンソールログを確認 → 内容を共有いただければ修正します。
- ロジック（計算・保存）はユニットテストで担保済みなので、UI配線/表示まわりが中心の想定。
